# Are-Birth-Rates-Really-Declining-A-Comparison-of-Nebraska-and-National-Birth-Trends-2020-2024-

Are Birth Rates Really Declining?
Nebraska vs U.S. Birth Trends (2020–2024)

Author: Linh Huynh
Course: DSC680 – Applied Data Science, Bellevue University

1. Project Overview

This project analyzes birth trends in Nebraska compared with national trends in the United States from 2020–2024.

The goal is to understand whether birth rates are truly declining and to help hospitals plan maternity services using data-driven insights rather than relying only on national averages. 


2. Dataset

CDC WONDER Natality Database

The dataset includes aggregated birth data such as:

Birth counts

State of residence

Average age of mother

Prenatal visits

Education level

Birth weight

for the years 2020–2024. 



3. Methods

The analysis includes:

Data cleaning and preparation

Trend comparison between Nebraska and the U.S.

Correlation analysis of maternal characteristics

Predictive modeling

Models used:

Linear Regression

Random Forest

4. Key Findings

Birth trends in Nebraska closely follow national trends.

Birth counts remained relatively stable from 2020–2024.

The average age of mothers is increasing over time.

Random Forest models performed better than linear regression for prediction.
